# ClinOva - Nursing Practical Assessment App
This is the FrontEnd of a Nursing Practical Assessment App (ClinOva), which is still in development.

The main aim of this app is to make practicals assesment at Nursing Training Colleges easy, fast and efficient.

## Tech Stack
- NextJs - Frontend framework
- Lucid React - Icons
- Shadcn - UI utility
- Django - Backend
